Documentación de io.github.piratemajo.magmalib.MagmaLib

La solución profesional para migrar plugins de Paper a Folia con una API moderna, type-safe y eficiente

Inicio Rápido

3 pasos para integrar io.github.piratemajo.magmalib.MagmaLib:
  1. Añadir como dependencia en tu pom.xml o build.gradle
  2. Llamar io.github.piratemajo.magmalib.MagmaLib.init(this) en onEnable()
  3. Reemplazar Bukkit.getScheduler() por io.github.piratemajo.magmalib.MagmaLib.task()
// 1. En tu plugin principal
@Override
public void onEnable() {
    io.github.piratemajo.magmalib.MagmaLib.init(this);

    // 2. Usar Task Builder
    io.github.piratemajo.magmalib.MagmaLib.task(() -> {
        player.sendMessage("¡Hola desde io.github.piratemajo.magmalib.MagmaLib!");
    })
    .at(player.getLocation())
    .afterTicks(20)
    .handleException(e -> getLogger().warning("Error: " + e.getMessage()))
    .run();
}

Inicializa la librería. Llamar en onEnable().

public void onEnable() {
    io.github.piratemajo.magmalib.MagmaLib.init(this);
}

boolean isFolia()

Detecta automáticamente si el servidor es Folia.

if (io.github.piratemajo.magmalib.MagmaLib.isFolia()) {
    // Código optimizado para Folia
}

Task Builder

task()

TaskBuilder task(Runnable runnable)

Punto de entrada para configurar tareas.

io.github.piratemajo.magmalib.MagmaLib.task(() -> {
    player.sendMessage("¡Hola!");
});

at()

TaskBuilder at(Location location)

Ejecuta en la región del chunk (Folia) o hilo principal (Paper).

.at(player.getLocation())

with()

TaskBuilder with(Entity entity)

Vincula la tarea al scheduler de la entidad (Folia).

.with(villager)

after()

TaskBuilder after(long delay, TimeUnit unit)

Retraso con cualquier unidad temporal.

.after(2, TimeUnit.SECONDS)

afterTicks()

TaskBuilder afterTicks(long ticks)

Retraso en ticks de Minecraft (1 tick = 50ms).

.afterTicks(20) // 1 segundo

every()

TaskBuilder every(long period, TimeUnit unit)

Configura tarea repetitiva.

.every(5, TimeUnit.SECONDS)

everyTicks()

TaskBuilder everyTicks(long ticks)

Intervalo repetitivo en ticks.

.everyTicks(100) // Cada 5 segundos

handleException()

TaskBuilder handleException(Consumer<Exception> handler)

Manejador personalizado de errores.

.handleException(e -> {
    plugin.getLogger().warning("Error: " + e.getMessage());
})

run()

Task run()

Ejecuta la tarea y devuelve objeto para manejo.

Task task = builder.run();
// Luego: task.cancel();

Funciones Avanzadas

runWithRetry()

void runWithRetry(Runnable task, int maxAttempts, long delay, TimeUnit unit)

Ejecuta con reintentos automáticos usando scheduler (NO threads manuales).

io.github.piratemajo.magmalib.MagmaLib.runWithRetry(() -> {
    connectToDatabase();
}, 3, 5, TimeUnit.SECONDS);

runTimerUntil()

Task runTimerUntil(Runnable task, long periodTicks, BooleanSupplier stopCondition)

Ejecuta periódicamente hasta que se cumpla una condición.

io.github.piratemajo.magmalib.MagmaLib.runTimerUntil(() -> {
    player.sendTitle("Cargando...", "", 10, 40, 10);
}, 20, () -> !player.isOnline());

callSync()

<T> CompletableFuture<T> callSync(Supplier<T> supplier)

Ejecuta en hilo principal y retorna valor con CompletableFuture.

io.github.piratemajo.magmalib.MagmaLib.callSync(() -> {
    return player.getInventory().getContents();
}).thenAccept(items -> {
    // Procesar items
});

forAllPlayers()

Itera jugadores con manejo seguro de errores y nulls.

Procesa chunks cargados (ejecuta asíncrono por defecto).

io.github.piratemajo.magmalib.MagmaLib.forAllLoadedChunks(chunk -> {

ticksToMs() / msToTicks()

long ticksToMs(long ticks) | long msToTicks(long ms)

Utilidades de conversión de tiempo.

Nota: io.github.piratemajo.magmalib.MagmaLib detecta automáticamente el tipo de servidor y usa el scheduler apropiado. Tu código funciona en ambos sin modificaciones.

Manejo de Entidades

Mejores Prácticas

Siempre usa .with(entity) para operaciones con entidades en Folia:

  • ✅ Ejecución en el scheduler correcto de la entidad
  • ✅ Cancelación automática si la entidad se elimina
  • ✅ Prevención de race conditions en Folia
  • ✅ Fallback seguro a Paper si no es Folia

Ejemplo Básico

io.github.piratemajo.magmalib.MagmaLib.task(() -> {
    if (entity.isValid()) {
        entity.setCustomName("Procesado");
    }
})
.with(entity)
.run();

Validación Avanzada

io.github.piratemajo.magmalib.MagmaLib.task(() -> {
    if (!entity.isValid() || entity.isDead()) return;
    entity.setGlowing(true);
})
.with(entity)
.cancelIf(() -> !player.isOnline())
.handleException(e -> {
    plugin.getLogger().warning("Error: " + e.getMessage());
})
.run();
