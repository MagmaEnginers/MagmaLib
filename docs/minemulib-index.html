MinemuLib v1.0

Documentación Completa de MinemuLib

La solución profesional para migrar plugins de Paper a Folia con una API moderna y eficiente

Inicialización

init()

MinemuLib.init(Plugin plugin)

Inicializa la librería con tu plugin principal.

@Override
public void onEnable() {
    MinemuLib.init(this);
}

isFolia()

boolean isFolia()

Determina si el servidor está ejecutando Folia.

if (MinemuLib.isFolia()) {
    // Código específico para Folia
}

Task Builder

task()

TaskBuilder task(Runnable runnable)

Crea una nueva tarea con el Runnable especificado.

MinemuLib.task(() -> {
    player.sendMessage("¡Hola!");
});

at()

TaskBuilder at(Location location)

Ejecuta la tarea en la ubicación especificada (vinculado al chunk).

.at(player.getLocation())

with()

TaskBuilder with(Entity entity)

Vincula la tarea a una entidad específica.

.with(villager)

after()

TaskBuilder after(long delay, TimeUnit unit)

Retrasa la ejecución con la unidad temporal especificada.

.after(2, TimeUnit.SECONDS)

every()

TaskBuilder every(long period, TimeUnit unit)

Configura la tarea como repetitiva con el intervalo especificado.

.every(5, TimeUnit.SECONDS)

async()

TaskBuilder async()

Ejecuta la tarea de forma asíncrona (fuera del hilo principal).

.async()

cancelIfUnloaded()

TaskBuilder cancelIfUnloaded(boolean cancel)

Cancela la tarea si el chunk no está cargado (solo para ubicaciones).

.cancelIfUnloaded(true)

cancelIf()

TaskBuilder cancelIf(BooleanSupplier condition)

Cancela la tarea si se cumple la condición especificada.

.cancelIf(() -> !entity.isValid())

handleException()

TaskBuilder handleException(Consumer<Exception> handler)

Manejador personalizado para excepciones durante la ejecución.

.handleException(e -> {
    player.sendMessage("Error: " + e.getMessage());
})

run()

Task run()

Ejecuta la tarea configurada y devuelve un objeto Task para manejo.

Task task = builder.run();

Nuevas Funciones

runSync()

void runSync(Runnable runnable)

Ejecuta una tarea en el hilo principal de forma segura.

MinemuLib.runSync(() -> {
    player.sendMessage("Ejecutado en el hilo principal");
});

forAllPlayers()

void forAllPlayers(Consumer<Player> action)

Ejecuta una acción para todos los jugadores en línea con manejo seguro de errores.

MinemuLib.forAllPlayers(player -> {
    player.sendTitle("¡Evento importante!", "", 10, 40, 10);
});

forAllLoadedChunks()

void forAllLoadedChunks(Consumer<Chunk> action)

Procesa todos los chunks cargados en todos los mundos.

MinemuLib.forAllLoadedChunks(chunk -> {
    // Optimizar chunks o realizar operaciones
});

runWithRetry()

void runWithRetry(Runnable task, int maxAttempts, long delay, TimeUnit unit)

Ejecuta una tarea con reintentos en caso de fallo.

MinemuLib.runWithRetry(() -> {
    connectToExternalService();
}, 3, 5, TimeUnit.SECONDS);

runTimerUntil()

Task runTimerUntil(Runnable task, long periodTicks, BooleanSupplier stopCondition)

Ejecuta una tarea periódica hasta que se cumpla una condición.

MinemuLib.runTimerUntil(() -> {
    player.sendMessage("Esperando...");
}, 20, () -> player.isOnline());

callSync()

<T> CompletableFuture<T> callSync(Supplier<T> supplier)

Ejecuta una tarea en el hilo principal que devuelve un valor.

MinemuLib.callSync(() -> {
    return player.getInventory().getContents();
}).thenAccept(items -> {
    // Procesar items
});

Manejo de Entidades

Buenas Prácticas con Entidades

Al trabajar con entidades, siempre usa .with() para garantizar:

  • Ejecución en el scheduler correcto de la entidad
  • Cancelación automática si la entidad se elimina
  • Optimización de rendimiento en Folia

Ejemplo Básico

MinemuLib.task(() -> {
    if (entity.isValid()) {
        entity.setCustomName("Procesado");
    }
})
.with(entity)
.run();

Tarea Repetitiva

MinemuLib.task(() -> {
    if (zombie.isValid()) {
        zombie.setTarget(player);
    }
})
.with(zombie)
.every(1, TimeUnit.SECONDS)
.run();

Validación Avanzada

Combina validaciones con manejo de errores:

MinemuLib.task(() -> {
    if (!entity.isValid() || entity.isDead()) return;
    
    // Operación segura
    entity.setGlowing(true);
})
.with(entity)
.handleException(e -> {
    plugin.getLogger().warning("Error con entidad: " + e.getMessage());
})
.cancelIf(() -> !player.isOnline())
.run();

Métodos de Utilidad

runNextTick()

void runNextTick(Runnable runnable)

Ejecuta una tarea en el próximo tick del servidor.

MinemuLib.runNextTick(() -> {
    player.sendMessage("Cargado!");
});

runLater()

void runLater(Runnable runnable, long delay, TimeUnit unit)

Ejecuta una tarea después de un retardo específico.

MinemuLib.runLater(() -> {
    player.sendMessage("Después de 5 segundos");
}, 5, TimeUnit.SECONDS);

runAsync()

CompletableFuture<Void> runAsync(Runnable runnable)

Ejecuta una tarea asíncrona con soporte para CompletableFuture.

MinemuLib.runAsync(() -> {
    // Proceso pesado
}).thenRun(() -> {
    // Al completar
});

callAsync()

<T> CompletableFuture<T> callAsync(Supplier<T> supplier)

Ejecuta una tarea asíncrona que devuelve un valor.

MinemuLib.callAsync(() -> {
    return fetchFromDatabase();
}).thenAccept(result -> {
    player.sendMessage("Datos: " + result);
});

executeIfLoaded()

void executeIfLoaded(Location location, Runnable task)

Ejecuta una tarea solo si el chunk en la ubicación está cargado.

MinemuLib.executeIfLoaded(block.getLocation(), () -> {
    block.setType(Material.DIAMOND_BLOCK);
});

Manejo de Tareas

cancel()

void cancel()

Cancela una tarea en ejecución (especialmente útil para tareas repetitivas).

Task task = MinemuLib.task(...).every(...).run();

// Al desactivar el plugin
task.cancel();

isCancelled()

boolean isCancelled()

Verifica si una tarea ha sido cancelada.

if (!task.isCancelled()) {
    // La tarea sigue activa
}

isRunning()

boolean isRunning()

Verifica si una tarea está actualmente en ejecución.

if (task.isRunning()) {
    // La tarea está activa
}
Mejor práctica: Siempre almacena referencias a tareas repetitivas y cancélalas en onDisable().

Migración de Paper a Folia

Tarea simple

// Paper
Bukkit.getScheduler().runTask(plugin, runnable);

// MinemuLib
MinemuLib.task(runnable).run();

Tarea repetitiva

// Paper
BukkitTask task = Bukkit.getScheduler()
    .runTaskTimer(plugin, runnable, 0, 20);

// MinemuLib
Task task = MinemuLib.task(runnable)
    .everyTicks(20)
    .run();

Operación con entidad

// Paper (peligroso en Folia)
Bukkit.getScheduler().runTask(plugin, () -> {
    entity.setCustomName("Nombre");
});

// MinemuLib (seguro)
MinemuLib.task(() -> entity.setCustomName("Nombre"))
    .with(entity)
    .run();

Tarea asíncrona

// Paper
Bukkit.getScheduler().runTaskAsynchronously(plugin, runnable);

// MinemuLib
MinemuLib.task(runnable).async().run();
